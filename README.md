## Setup steps

- **Clone the repo**
- **Run composer install**
- **Run npm install**
- **Copy .env.example to .env and fill in database and Google Map API details**
- **Run php artisan key:generate**
- **Run php artisan migrate**
- **Run php artisan storage:link**
- **Run php artisan db:seed**
